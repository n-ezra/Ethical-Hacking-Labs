# 🌐 WHOIS Lookup – Information Gathering

## 📌 Introduction
WHOIS Lookup is a **passive reconnaissance technique** used to gather information about a domain or IP address. It helps identify ownership, registration details, and DNS infrastructure.

---

## 🎯 Objective
- Identify domain ownership  
- Collect registration details  
- Find DNS and hosting information  
- Support reconnaissance phase  

---

## 🛠️ Tools Used
- Kali Linux (whois tool)
- Whois.com (online lookup)
- MXToolbox

---

## ⚙️ Command Syntax
```bash
whois <target-domain>
```

### Example:
```bash
whois vulnweb.com
```

---

## 📊 Output Details
WHOIS results include:

- Domain Name  
- Registrar  
- Creation Date  
- Expiry Date  
- Registrant Information  
- Name Servers  
- Domain Status  

---

## 🔍 Types of WHOIS Lookup

### 1. Domain WHOIS
- Provides domain registration details  

### 2. IP WHOIS
- Identifies ISP / organization  

---

## 🧠 Use Cases
- Footprinting & Reconnaissance  
- Cybersecurity investigations  
- Threat intelligence  
- Domain verification  

---

## ⚠️ Limitations
- Data may be hidden (privacy protection)  
- Not always accurate  
- Some details are masked  

---

## 📸 Practical Steps

1. Open Kali Linux terminal  
2. Run:
   ```bash
   whois vulnweb.com
   ```


---

## 🚀 Conclusion
WHOIS Lookup is a **basic but powerful reconnaissance technique** used in the footprinting phase of ethical hacking.
